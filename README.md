# Island Simulation

## Описание
Симуляция экосистемы острова с животными.

## Технологии
Java, Maven, JUnit 5

## Реализованные тесты

### Unit тесты (10)
- проверка создания животных
- проверка принадлежности к типу (хищник/травоядное)
- проверка веса

### Functional тесты (2)
- хищник ест травоядного
- хищник не ест хищника

## Запуск тестов

mvn test
